# Proyecto Vite + React + TypeScript

Este proyecto usa Vite para un entorno de desarrollo rápido con React y TypeScript. Si recibías errores al ejecutar `npm start`, ahora existe un alias que levanta el servidor de desarrollo correctamente.

## Requisitos
- Node.js 18 o superior
- npm 9 o superior

## Instalación
```bash
npm install
```

## Ejecución en desarrollo
```bash
npm start
```
O, si prefieres el comando original de Vite:
```bash
npm run dev
```

## Compilación para producción
```bash
npm run build
```

## Vista previa del build
```bash
npm run preview
```
